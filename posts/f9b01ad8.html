🚀 3.1逻辑回归

🌟 学习目标

  • 了解逻辑回归的现实应用场景
  • 掌握逻辑回归的核心原理
  • 理解损失函数与优化算法的数学逻辑

🌐 1 逻辑回归的应用场景

生活中常见的二分类问题都可以用逻辑回归解决：

  • ✉️ 垃圾邮件识别（是 / 否）
  • ❤️ 疾病诊断（患病 / 健康）
  • 💰 金融风控（欺诈 / 正常）
  • 📲 账号检测（虚假 / 真实）
  • 📈 广告效果（点击 / 不点击）

特点：所有问题都只有两个对立结果，逻辑回归通过概率判断归属。

🧠 2. 逻辑回归的核心原理

🔄 输入层：线性回归的输出

🚦 激活函数

  • 🐱sigmoid 函数

  • 判断标准

    • 回归的结果输入到sigmoid函数当中
    • 输出结果：[0, 1]区间中的一个概率值，默认为0.5为阈值

📊 分类决策：概率阈值

默认以 0.5 为分界：

  • P (y=1) ≥ 0.5 → 预测为正类（1）
  • P (y=1) < 0.5 → 预测为负类（0）

📌 阈值可调：根据业务需求调整（如风控可设为 0.7）

📉 3. 损失函数与优化算法

🔥 损失函数：对数似然损失

单个样本损失：

$

\(L(y, \hat{y}) = \begin{cases} -\ln(\hat{y}) & \text{如果 } y=1 \\ -\ln(1-\hat{y}) & \text{如果 } y=0 \end{cases}\)**整体损失**：

\(J(w) = -\frac{1}{m}\sum_{i=1}^m [ y^{(i)}\ln(\hat{y}^{(i)}) + (1-y^{(i)})\ln(1-\hat{y}^{(i)}) ]\)**损失函数特性**：

  • 当预测正确时，损失趋近于0
  • 当预测错误时，损失趋近于无穷大

⚙️ 优化算法：梯度下降

  1. 计算梯度：\(\frac{\partial J}{\partial w_j} = \frac{1}{m}\sum_{i=1}^m (\hat{y}^{(i)} - y^{(i)})x_j^{(i)}\)
  2. 更新参数：\(w_j = w_j - \alpha \frac{\partial J}{\partial w_j}\)

💡 为什么不用均方误差？ 因为逻辑回归的输出是概率值，均方误差会导致非凸问题，难以优化。

📝 4. 总结对比表

对比项线性回归逻辑回归
任务类型回归问题分类问题
输出范围任意实数[0,1]区间概率值
激活函数无Sigmoid函数
损失函数均方误差对数似然损失
优化目标最小化预测误差最大化分类概率

🎯 实践建议

  1. 特征预处理：标准化/归一化
  2. 处理数据不平衡：调整阈值或使用SMOTE算法
  3. 评估指标：准确率、AUC-ROC曲线

通过逻辑回归，我们实现了从线性回归到分类模型的跨越，后续可以尝试使用Python的sklearn.linear_model.LogisticRegression进行实战！
